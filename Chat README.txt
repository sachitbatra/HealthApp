Chat Application

A new thread object is created when a user enters the URL /consultations/other_user_mail

Once, this thread is created, it shows up in both the users' inbox page accessible at /consulations
Inbox page simply lists all the threads that exist for the user.

The messages are saved along with the thread object to which they are being added along with the sender of the message.

Websockets are used to send messages asynchronously.
